# AndThatsOnCrypto 🚀

Welcome to the official repository for **AndThatsOnCrypto**, a project dedicated to demystifying the world of cryptocurrency for beginners. This repository houses educational web resources designed to be clear, accessible, and easy to navigate.

## 📖 About This Project
The goal of this project is to provide a "no-nonsense" entry point for anyone curious about crypto. Originally inspired by the mission of our Facebook community, these web pages serve as a permanent, searchable home for essential crypto knowledge.

## 📂 Current Resources
* **Introduction to Crypto**: A comprehensive one-pager covering everything a newcomer needs to know to get started safely.
    * **Live Link:** `https://www.AndThatsOnCrypto.com/introduction/`

## 🛠 Project Structure
To keep things organized, each major topic is hosted in its own folder:
* `/introduction` - Contains the primary "Introduction to Crypto" guide.
* More modules coming soon!
